Frequently Asked Questions

Answers for admins and members using Archivist in Discord.

How do I add Archivist to my Discord server?

Use the bot invite link from the homepage and approve requested scopes and permissions. After joining, verify channel permissions and run /docs add to upload your first PDF.

What are the default limits?

Current default limits are 3 indexed documents per server, 5 MB max per PDF, and 50 queries per UTC day.

Can we configure document retention?

Yes. Admins can use /docs retention set to define server retention in hours. Valid range is 48 hours to 8760 hours (one year).

How does privacy work?

Archivist is built around server-level data isolation and explicit retention behavior. Read full commitments on the Privacy Policy page.

What if my PDF is scanned and not machine-readable?

OCR fallback is plan-aware. Free-tier servers receive an upgrade prompt when OCR is required, while supported paid tiers can use OCR fallback automatically.

What should I do if answers look wrong?

Check document ingestion status with /docs status, verify the file is indexed, and review quota state via /usage. If needed, contact support details listed on the Contact page.
