What Archivist does

Document-grounded answers

Answers include references to the indexed file and page so users can verify each claim quickly.

Privacy-first architecture

Archivist is designed for server-local data isolation with retention controls and explicit limits.

Fast command workflow

Use slash commands for upload, retention, plan management, and usage visibility without leaving Discord.

How it works

  1. Admins upload a PDF with /docs add.
  2. Archivist indexes text and tracks ingestion status.
  3. Members ask with /ask or @Archivist ... and get citation-backed responses.
