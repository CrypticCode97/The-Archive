Reference

Archivist Command Guide

Use this page as your quick command reference for setup, querying, retention policy, and plan controls.

Core query commands

/ask

Ask a question against indexed documents.

Example: /ask question:"what does this policy mean?"

@Archivist ...

Mention-based query in channel, for quick follow-ups.

Example: @Archivist compare probation and contractor rules

Document management commands

/docs add

Upload one PDF (maximum 5 MB). Requires Manage Server.

/docs list and /docs status

List indexed files and review ingestion status per document.

/docs remove

Delete a document from index using UUID and confirmation token. Requires Manage Server.

Retention and plan commands

/docs retention status

Shows current retention policy for the server.

/docs retention set

Sets retention in hours (48 to 8760). Requires Manage Server.

/plan status, /plan upgrade, /plan manage

Plan visibility and upgrade path. Upgrade requires Manage Server.

/usage

Shows plan-aware usage counters and limit posture.

Permission notes

Command group Permissions Who should use it
Ask and mention queries Standard channel access All members
Docs add/remove and retention set Manage Server Admins and moderators
Plan upgrade Manage Server Server owners/admins
